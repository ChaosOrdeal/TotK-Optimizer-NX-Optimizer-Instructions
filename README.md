# TotK-Optimizer-NX-Optimizer-Instructions
A one-page document to help with installation/use.

- NXO creates a mod that applies its settings, and an additional cheat mod, if used
- - these mods are automatically placed in the proper folder, and are [de-]activated normally
- NXO can be run from almost anywhere; do not copy any of it into your mods folder
- run NXO after installing an applicable emulator and game, while the emu is not running
- if NXO seems to be ignoring new settings, close it, delete the .ini in NXO's folder; re-run NXO and re-apply both the Manager and the Cheat mods
- change options and cheats all you want and Apply [both] again to use your latest settings
